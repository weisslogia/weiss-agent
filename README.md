# My CLI Tool

A Node.js terminal application built with TypeScript.

## Installation

```bash
npm install
npm run build
```

## Usage

```bash
npm start -- hello --name "Your Name"
npm run dev -- hello --name "Your Name"
```

## Available Commands

- `hello` - Prints a greeting message

## Development

```bash
# Run in development mode
npm run dev

# Run tests
npm test

# Build for production
npm run build

# Lint code
npm run lint
```
